1. To initialise frontend 
	1. Go to frontend folder 
	2. open the folder in the terminal
	3.execute these commands one after other (npm i,npm run dev)
	4.This runs the development server on localhost:5173 
	5. Or else you can run npm run build which will give you html,css,js files you can also use them
2.To initialise Backend
	1.go to backend folder
	2.open backend folder in terminal
	3.change the mongodbURL form the .env file to your mongodb URL
	3.execute these commands one after other (npm i,nodemon server.js or node server.js)
	4.This starts the nodejs server on port 3000
